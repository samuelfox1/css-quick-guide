# Descendant Combinator

- Selecting elements that are nested within other HTML elements, also known as descendants.
```
<ul class='main-list'>
  <li> one </li>
  <li> two </li>
  <li> three </li>
</ul>
```
- select all li decendants of the element with a class name of main-list 
```
.main-list li {
    property: value;
}
```
